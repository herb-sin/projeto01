Sobre

Esse site é o primeiro projeto realizado em grupo da Cocreare Academy

A idéia desse projeto é criar página web responsiva para visualização de mensagens e imagens com integração com a API do Twitter.

A Cocrere Academy é especialista em desenvolvimento web/mobile, apaixonados por compartilhar conhecimento, vivem para tecnologia e respiram criatividade e inovação. Oferecem serviços como desenvolvimento de sites, aplicativos, sistemas web e outsourcing de desenvolvimento.
