Projeto


Esse site é o primeiro projeto realizado em grupo da Cocreare Academy

A idéia desse projeto é criar página web responsiva para visualização de mensagens e imagens com integração com a API do Twitter.

Nesse projeto trabalhamos com HTML, CSS, JavaScript e Angular. Foi um projeto desafiador e cheio de conquistas.
